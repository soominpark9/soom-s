CSS - diaplay속성에 대해 알아보자!

  • 요소를 화면에 어떻게 보여줄지 결정하는 속성
  • html 태그는 block과 inline 요소로 나눔

inline

  1. ex : span,a,img
  2. content(내용물) 만큼 공간을 할당 -> width, height, padding, margin 을 사용할 수 없다
  3. 줄 바꿈 없이 바로 옆에 배치(기본적으로)
d-inline1 클래스를 갖는 div
d-inline2

block

  1. ex: div, p, ul, ol
  2. 한 줄을 다 차지함 -> width, height, padding, margin 사용이 자유롭다 ( 다 적용됨 )
d-block1 클래스를 갖는 div
d-block2 클래스를 갖는 div

inline-block

  1. block 요소 처럼 크기 지정을 하되
  2. inline 요소 처럼 다른 내용들과 같은 줄을 공유 하고 싶을 경우 사용한다
  3. 블록과 인라인 요소의 장점을 가진다
d-inline-block1 클래스를 갖는 div
d-inline-block1 클래스를 갖는 div

none

  1. 요소를 화면에 표시하지 않음
  2. 주의 ) display: none ; 요소가 차지하는 공간도 사라짐
  3. 요소를 그리지 않음 ( 랜더링 하지 않는다 = 그리지 않는다 )
